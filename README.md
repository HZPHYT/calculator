# 多功能计算器小程序

基于 uni-app + uniCloud 开发的多功能计算器微信小程序。

## 功能列表

### 基础计算器
- 四则运算：加、减、乘、除
- 高级运算：百分比(%)、平方(x²)、开根号(√)
- 括号支持：支持复杂表达式
- 历史记录：自动保存计算历史，支持批量删除

### 科学计算器
- 三角函数：sin、cos、tan（支持角度制）
- 对数函数：log（常用对数）、ln（自然对数）
- 其他函数：exp、√、^
- 常量：π、e
- 完整括号支持

### 专业计算器
- BMI计算器：身体质量指数计算
- 日期计算器：日期天数差计算
- 时间戳转换器：时间戳与日期互转
- 个税计算器：个人所得税计算
- 房贷计算器：等额本息/等额本金月供计算
- 字数统计器：字符数、行数统计
- 单位换算器：长度、重量、面积、温度、体积

## 技术栈

- **框架**：uni-app (Vue3)
- **计算引擎**：mathjs
- **样式**：SCSS
- **数据存储**：本地存储 (uni.getStorageSync)

## 开发运行

```bash
# 安装依赖
npm install

# 运行微信小程序开发模式
npm run dev:mp-weixin

# 发行小程序
npm run build:mp-weixin
```

## 使用说明

### 普通计算器
- 按 `√` 再按数字开根号：如 `√9=3`
- 按数字再按 `x²` 平方：如 `5x²=25`
- 按 `%` 直接得到小数：如 `20%=0.2`

### 科学计算器
- 三角函数使用角度制：如 `sin30=0.5`
- 支持复杂表达式：如 `sin45+cos60`
